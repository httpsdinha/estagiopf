Instale o pyinstaller:

pyinstaller é uma ferramenta popular para criar executáveis a partir de scripts Python.
Instale-o usando o pip:
	pip install pyinstaller
Crie o Executável:

Navegue até o diretório onde seu script Python está localizado.

Execute o seguinte comando para criar um executável:
	pyinstaller --onefile --noconsole seu_script.py

Substitua seu_script.py pelo nome do seu arquivo Python.
